# Movie & TV Show Recommender System

Proyecto final del curso de Big Data y Business Intelligence.

## Objetivo
Desarrollar un sistema recomendador de películas y series basado en contenido, utilizando información como descripciones, géneros y reparto.

## Dataset
Datos obtenidos de Kaggle (Netflix, HBO, Disney+), con información de títulos y créditos.

## Metodología
Sistema recomendador basado en similitud de contenido (TF-IDF + cosine similarity).

## Estado del proyecto
En desarrollo.
